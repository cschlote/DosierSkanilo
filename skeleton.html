Welcome to DosierSkanilo

DosierSkanilo is a blob-centric scanner for media, archives, and torrent metadata. It computes checksums, extracts signatures, and persists results as JSON for duplicate detection and analysis workflows.

Release 26.6.x Tag v26.6.0 Notes CHANGELOG Docs Build runtime Compiler ldc2 (default) Theme Slate

Core Capabilities

  • Checksum pipeline: MD5, SHA1, XXH64
  • MediaInfo + file utility signatures
  • Archive entry inspection (`zip`, `tar`, `rar`, `7z`)
  • Torrent metadata extraction with magnet/info-hash
